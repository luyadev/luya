# Multiple inputs

Generate a expandable list with custom nested fields and/or plugins inside each row.

### Definition in block

```php
['var' => 'people', 'label' => 'People', 'type' => self::TYPE_MULTIPLE_INPUTS, 'options' => [
        [
            'type' => self::TYPE_SELECT,
            'var' => 'salutation',
            'label' => 'Salutation',
            'options' => [
                ['value' => 1, 'label' => 'Mr.'],
                ['value' => 2, 'label' => 'Mrs.'],
            ]
        ],
        [
            'type' => self::TYPE_TEXT,
            'var' => 'name',
            'label' => 'Name'
        ],
    ]
]
```
